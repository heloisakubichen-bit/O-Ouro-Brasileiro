# O Ouro Brasileiro ☕

Um site educativo sobre a história e cultivo do café no Brasil.

## 📖 Conteúdo

O site aborda os seguintes tópicos:

- **História do Café no Brasil**: Como o café chegou ao Brasil e se tornou o "ouro brasileiro"
- **Plantação de Café**: Informações sobre escolha de terreno, clima e solo ideal
- **Cultivo e Colheita**: Ciclo de vida da planta, floração, frutificação e colheita
- **Importância Econômica**: O papel do café na economia brasileira

## 🎨 Design

O site utiliza uma paleta de cores baseada em tons de marrom claros, evocando a cor do café:

- **Marrom Principal**: #8B6F47
- **Marrom Claro**: #D4AF9F
- **Marrom Mais Claro**: #E8D4C4
- **Marrom Escuro**: #5C4A3D
- **Marrom Acentuado**: #C4955F

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/heloisakubichen-bit/o-ouro-brasileiro.git
   ```

2. Abra o arquivo `index.html` no seu navegador ou use um servidor local:
   ```bash
   python -m http.server 8000
   ```

3. Acesse `http://localhost:8000` no seu navegador

## 📱 Responsividade

O site é totalmente responsivo e foi otimizado para:
- Desktop (1920px+)
- Tablet (768px - 1024px)
- Mobile (até 767px)
